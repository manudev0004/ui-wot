ui-wot — Event-driven integration example

This page demonstrates listening to typed DOM events emitted by components (e.g., valueChange, dateChange).

Note: Start the components dev server first so the web components are available. From the repository root run:

cd packages/components
npm start

Then open this file in a browser (or serve it from a local static server). The page expects the dev server to expose the component bundle at http://localhost:3333/build/ui-wot-components.esm.js. If your dev server uses a different port/path, update the <script> tag below.
