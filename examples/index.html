Getting Started

Include the script and start using the components in any HTML page.

<script type="module" src="./build/ui-wot-components.esm.js"></script>

<ui-number-picker value="5" label="Quantity"></ui-number-picker>
<ui-toggle label="Switch"></ui-toggle>

Device Integration

Connect to IoT Devices

Use Thing Description URLs to connect directly to devices via HTTP, CoAP, or MQTT.

HTTP Device

<ui-number-picker
  td-url="http://device.local/temp"
  protocol="http"
  mode="readwrite">
</ui-number-picker>

Read-Only Sensor

<ui-number-picker
  td-url="http://sensor.local/temp"
  mode="read"
  label="Temperature">
</ui-number-picker>

Toggle Device

<ui-toggle
  td-url="http://light.local/power"
  protocol="http"
  mode="readwrite">
</ui-toggle>

Custom Events

Number Picker with Callback

Value: 5
window.handleNumberChange = function(data) {
  console.log('Value:', data.value);
};

<ui-number-picker
  on-change="handleNumberChange">

Toggle with Callback

State: false
window.handleToggleChange = function(data) {
  console.log('State:', data.active);
};

<ui-toggle
  on-change="handleToggleChange">
